# 🛍️ Retail Sales Performance Analysis

An end-to-end Business Analytics project that analyzes retail sales data using **MySQL** and **Power BI**. The project focuses on evaluating sales performance, profitability, customer behavior, and product performance to generate actionable business insights.

---

## 📖 Project Overview

This project analyzes four years of retail sales data to help business stakeholders understand overall business performance.

SQL was used to perform business analysis and calculate key performance indicators (KPIs), while Power BI was used to build interactive dashboards for data-driven decision-making.

---

## 🎯 Business Problem

Retail businesses generate large volumes of sales data every day. Without proper analysis, it becomes difficult for management to identify:

- Which regions perform the best
- Which products generate the highest revenue and profit
- Which customers contribute the most value
- Which products or locations generate losses
- How sales performance changes over time

This project answers these business questions using SQL analysis and interactive Power BI dashboards.

---

## 🎯 Project Objectives

- Analyze overall business performance using key performance indicators (KPIs).
- Identify top-performing and underperforming regions, states, products, and customers.
- Measure profitability across different business dimensions.
- Understand customer purchasing behavior and customer segment performance.
- Evaluate product performance using sales and profit analysis.
- Apply advanced SQL techniques to solve analytical business problems.
- Build interactive Power BI dashboards to support data-driven decision-making.

---

## 📂 Dataset Information

**Dataset Name:** Sample Superstore

**Time Period:** 2014–2017

The dataset contains retail sales transactions including:

- Orders
- Customers
- Products
- Categories
- Sales
- Profit
- Discount
- Shipping Details
- Region
- State
- City

The cleaned dataset was used for SQL analysis and Power BI dashboard development.
